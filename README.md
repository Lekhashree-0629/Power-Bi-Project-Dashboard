# Coffee Shop Sales Dashboard
## Power BI Interactive Sales Analysis
This repository contains an interactive Power BI Dashboard built to analyze and monitor the sales performance of a Coffee Shop business.
The dashboard provides insights across sales, orders, product categories, store locations, weekdays/weekends, and hourly trends.

## Dashboard Overview
### 1. Key Metrics
The top panel highlights the most important monthly KPIs:
-	Total Sales: $119K
-	Total Orders: 25,335
-	Total Quantity Sold: 36,469
-	MoM Comparison: Metrics show % increase or decrease vs. last month
